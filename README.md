# 📝 To-Do List API - Spring Boot

REST API for task management (To-Do List) built with **Java and Spring Boot**.  
The project implements full **CRUD operations** for creating, reading, updating, and deleting tasks.

---

## 🚀 Technologies Used

- Java 17+
- Spring Boot
- Spring Web
- Spring Data JPA
- Maven
- Database (H2, MySQL)
- Lombok

---

## 📌 Features

- ✅ Create a new task
- 📋 List all tasks
- 🔎 Get a task by ID
- ✏️ Update a task
- ❌ Delete a task

---

💻 Project developed for study purposes using **Spring Boot**.
